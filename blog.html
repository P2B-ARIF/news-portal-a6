    • 01-No Question => var, let & const এর পার্থক্য কি..?

    var দিয়ে ভ্যারিয়েবল ডিক্লেয়ার করলে সেইটা re-declare এবং re-assign দুইটাই করা যায় । let দিয়ে ভ্যারিয়েবল ডিক্লেয়ার করলে সেইটা re-assign করা যায় কিন্ত re-declare করা যায় নাহ এবং const দিয়ে ভ্যারিয়েবল ডিক্লেয়ার করলে সেইটা re-declare এবং re-assign কোনোটাই করা যায় নাহ ।


    02-No Question => Arrow Function & Regular Function পার্থক্য কি..?

    Arrow Function - একটি নতুন বৈশিষ্ট্য ES6-এ প্রবর্তিত হয়েছে - জাভাস্ক্রিপ্টে অ্যারো ফাংশনে মাধ্যমে সংক্ষিপ্ত ভাবে ফাংশন লিখা যায়, যদিও Regular Function এবং Arrow Function উভয় একইভাবে কাজ করে ।
    Regular Function - এর ক্ষেত্রে Function keyword ব্যবহার করতে হয় । এবং কোনো কিছু return করতে চাইলে return keyword ব্যবহার করতে হয় ।


    03-No Question => map(), forEach(), filter() & find() এর মধ্যে পার্থক্য কি..?

    1) .map() - loop চালিয়ে একটি অ্যারে return করে ।
    2) .filter() loop চালিয়ে যখন আমি একটি অ্যারে থেকে একাধিক উপাদানের মধ্যে একটি object বাহির করার জন্য ব্যবহার করি ।
    3) .find() loop চালিয়ে যদি একটি অ্যারে থেকে একটি উপাদান বাহির বা চেক করার জন্য find ব্যবহার করি ।
    4) .forEach() loop চালিয়ে যখন একটি Arrow এর একাধিক উপাদান থেকে একটি একক মান পেতে forEach ব্যবহার করি এটি কোনো কিছু return করে না ।


    04-No Question => Template Strings কেনো ব্যবহার করি..?

    Template Literals/ Strings ব্যবহার করার জন্য back-ticks (``) এর সাহায্যে আমরা জাভাস্ক্রিপ্টে html বা অন্যান্য কিছু লিখতে সাহায্য করে । এটি es6 version এর পর জাভাস্ক্রিপ্টে যুক্ত হয় ।
